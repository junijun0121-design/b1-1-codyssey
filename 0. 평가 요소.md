# 1. 업무 과업 정의
## 업무 과업
메일 작성 초안
## 타겟 사용자
온라인 쇼핑몰 고객지원 담당자
## 입력 규격
| 항목 | 자료형 | 필수 여부 | 설명 |
|--------|--------|--------|--------|
| 고객 문의 내용 | Text | 필수 | 고객이 남긴 문의 |
| 고객 이름 | Text | 선택 | 고객 식별 정보 |
| 고객 등급 | Text | 선택 | 일반/VIP 등 |
| 주문번호 | Text | 선택 | 주문 식별 정보 |
| 회사 정책 | Text | 필수 | CS 처리 기준 |
| 처리 결과 | Text | 필수 | 실제 조치 내용 |
| 톤 | Text | 선택 | 정중함, 공감형 등 |
## 출력 규격
| 항목 | 설명 |
|--------|--------|
| 제목 | 문의 유형에 맞는 이메일 제목 |
| 인사말 | 고객 이름 포함 가능 |
| 문의 답변 | 문의에 대한 설명 |
| 정책 안내 | 제공된 정책만 활용 |
| 후속 조치 | 환불, 교환, 배송 안내 등 |
| 마무리 인사 | 정중한 종료 문구 |

# 3. 3종 이상 모델 비교 결과표 및 선정 이유
| 평가 축 | GPT-4o | Claude Sonnet 4.6 | Gemini 3.5 Pro |
| :-: | :-: | :-: | :-: |
| 내용 정확성 | 9 | 10 | 7 |
| 메일 품질 | 10 | 8 | 9 |
| 환각 대응력 | 9 | 10 | 7 |
| 톤 조절 | 10 | 8 | 9 |
| 실무 활용성 | 10 | 9 | 10 |
| **총점** | **48** | **45** | **42** |

| 평가 축 | GPT-4o | Claude Sonnet 4.6 | Gemini 3.5 Pro |
| :-: | :-: | :-: | :-: |
| 내용 정확성 | 제공된 프롬프트만 활용 | 문단 별로 깔끔하게 제시 | 문맥 유지 및 환각 방지에 취약 |
| 메일 품질 | 자연스러운 비즈니스 메일 형식 | 장황한 표현 | 일부 문장이 어색 |
| 환각 대응력 | 제공된 정책만 활용 | 정책 준수 우수함 | 정책 제대로 확인 안 함 |
| 톤 조절 | 공감형·비즈니스형 전환 우수 | 공감 표현 우수 | 공감형·비즈니스형 전환 우수 |
| 실무 활용성 | 대화 및 아이디어 면에서 두각 | 분석적민 연에서 두각 | 사실 탐색에 유리 |

| 모델 | 근거 |
| :-: | :-: |
| GPT-4o | 회사 정책에 없는 쿠폰 보상 건을 거절하면서 VIP 고객 정보 꾸준히 유지 |
| Claude Sonnet 4.6 | 쿠폰 정책이 없는 사실을 얘기했으나 처리 건에 대해 장황한 느낌 |
| Gemini 3.5 Pro | 정책을 제대로 확인하지도 않고 환불 완료했다고 고객에게 제시 |

- 결론: '메일 초안 작성' 과업은 문서 정리 및 분석에 유리한 Claude나 최신 정보 및 사실 탐색에 유리한 Gemini 모델보다는, 대화 및 아이디어 면에서 두각을 드러내는 **GPT**에서 가장 점수가 높은 모델로 채택합니다. 고객이라는 상대방의 감정을 공감하는 점이 필요해 사용자와 AI의 대화 균형을 맞추는 것이 중요하다고 봤고, 여러 가지 상황을 AI에게 줬을 때 이에 대해 어떻게 대응해야 할지 의견을 많이 주다 보니 GPT가 특히 메일을 자연스럽게 작성하면서 추후 실무적인 면에서 활용도가 높다고 봤습니다.

# 4. Few-shot 예시 3개 이상 존재 및 모호한 입력 흐름 포함 여부
## 예시 1
- 문의 내용: 주문한 상품이 아직 배송되지 않아서 배송 현황을 알고 싶습니다.
- 회사 정책: 배송 지연 시 예상 도착일 안내 가능
- 처리 결과: 택배사 물류 지연으로 5일 뒤 배송 예정
- 전달 사항: 고객에 대한 사과 + 배송 현황 안내 + 예상 도착일 안내
## 예시 2
- 문의 내용: 배송이 5일이나 늦어져서 불편을 겪었는데 보상 받을 수 있나요?
- 회사 정책: 배송 지연 3일 경과 시, 배송비 환불 가능
- 처리 결과: 배송비 전액 환불 승인
- 전달 사항: 사과 + 환불 정책 안내 + 배송비 환불 승인 및 일정 안내
## 예시 3
- 문의 내용: “해당 상품 받았는데 환불해주세요.”
- 회사 정책: 미개봉 상품 한해 전액 환불 가능
- 되묻기: 번거로우시겠지만 상품 개봉 여부 또는 환불을 요청하시는 사유를 회신으로 알려주시면 관련 기준에 따라 빠르게 검토 후 안내드리겠습니다.
- 처리 결과: 환불 사유는 배송 지연으로 인한 단순 변심이고 해당 상품은 미개봉 상태이므로 전액 환불 가능
