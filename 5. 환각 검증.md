# 환각 검증

## 환각 검증 질문 및 판정 결과
| 질문 | AI 응답 | 실제 여부 | 판정 |
| - | - | - | - |
| 미개봉 상품 무조건 환불? | 예 | 정책 확인 필요 | Fail |
| 쿠폰 자동 지급? | 예 | 정책 없음 | Fail |
| 배송비 환불? | 3일 경과 시 가능 | 정책 일치 | Pass |
| 고객 정보 누락? | 추가 질문 | 추가 질문 | Pass |
| 법적 책임? | 법률 자문 아님 | 정상 | Pass |

## 환각 정의
환각(Hallucination)은 AI가 근거 없는 정보를 사실로 생성하는 현상입니다.

여기서 '창의적 생성'과 구분되는 이유는 환각은 실제로 존재하지 않는 정책, 규정 그리고 사실 등을 만들어내기 때문에 창의적 생성으로 볼 수 없습니다.

현재 진행된 메일 초안 작성에서는 고객 응대 업무 특성상 정확한 정보를 전달하는 것이 중요하기 때문에 환각을 최소화해야 합니다.
