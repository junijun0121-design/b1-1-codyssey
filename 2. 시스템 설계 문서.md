# 시스템 설계 문서

## 타겟 사용자
### 1. 업무 문제
- 배송 지연으로 인해 VIP 고객에게 환불 안내 관련 메일 초안 작성
- 프로젝트 내 회사 정책으로 배송 지연 3일 경과 시, 배송비 환불 가능
- 상품 미개봉 시, 배송비+상품 포함 전액 환불 가능

### 2. 타겟 사용자 (페르소나)
온라인 쇼핑몰 고객 지원팀 담당자 이OO

### 3. 입력 데이터 형태 (고객에게 받을 정보)
- 고객명
- 주문번호
- 배송 지연일
- 환불 사유
- 상품 개봉 여부
- 회사 정책

### 4. 출력 규격 (제출 형식)
이메일 형식의 PDF 파일

## 입력 템플릿
    [업무 과업]: 메일 초안 작성
    [주문번호]: 
    [문의 내용]: 
    [환불 사유]: 
    [상품 상태]: 
    [회사 정책]: 
    [추가 요청사항]: 

    [목적]: 배송 지연에 대한 고객 문의 ‘답변 메일’ 작성
    [대상]: 고객 김OO
    [핵심 포인트]: 배송 지연에 대한 사과
                  배송 지연 관련 환불 정책 안내
    [톤]: 공감하면서 비즈니스적 말투
    [금지어]: “어쩔 수 없다.”와 같은 고객에게 책임을 전가하는 표현이나 오해를 받을 수 있는 내용 불가
              회사 정책을 벗어나는 약속 불가
    [필수 포함]: 사과 문구 + 해결 방안 및 안내
